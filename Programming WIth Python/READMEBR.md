# 📁 Repositório de Códigos de Estudo

Este repositório foi criado com o objetivo de organizar e compartilhar códigos de estudo de forma clara e separada por propósitos. Os arquivos estão divididos em duas pastas principais:

---

## 🧠 Pastas Principais

### 📂 `resolution_problems`

Esta pasta contém a **resolução de problemas diversos** utilizando algoritmos e estruturas de dados. É ideal para quem está praticando lógica de programação ou se preparando para competições e entrevistas técnicas.

#### Exemplos de conteúdo:
- Problemas de ordenação
- Cálculo de média, soma, fatorial
- Estruturas de repetição e condicionais
- Desafios matemáticos e lógicos

> 📎 *Cada arquivo dentro desta pasta representa um problema resolvido com explicações em comentários.*

---

### 📂 `work_with_classes`

Esta pasta é dedicada ao **trabalho com Programação Orientada a Objetos (POO)**. Aqui, são demonstradas implementações de classes, objetos, herança, encapsulamento, polimorfismo e outros conceitos relacionados.

#### Exemplos de conteúdo:
- Criação de classes em diferentes linguagens
- Simulações com objetos
- Exercícios de POO
- Projetos com múltiplas classes

> 🧩 *Os arquivos geralmente contêm exemplos práticos de uso das classes criadas.*

---

## 🛠️ Como Usar Este Repositório

1. **Clone o repositório** em sua máquina:
```bash
git clone https://github.com/seu-usuario/nome-do-repositorio.git
```

2. **Acesse a pasta do projeto**:

```bash
cd nome-do-repositorio
```

3. **Escolha uma das pastas para explorar**:

```bash
cd resolution_problems
```

```bash
cd work_with_classes
```

4. **Execute os arquivos no ambiente da linguagem correspondente.**

```bash 
python nome_do_arquivo.py
```
