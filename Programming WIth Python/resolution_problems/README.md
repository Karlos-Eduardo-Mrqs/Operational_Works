## 🤝 How to Contribute

If you’d like to contribute to this repository, follow the steps below:

### ✅ Clone the Repository

```bash
git clone https://github.com/your-username/python-repository.git
```

## ✍️ Add Your Programs

Create a new folder or add your programs to the existing ones (`simple_problems` or `functions_problems`).

Update this `README.md` file with the description of your added program.

---

## 📤 Submit Your Changes

After adding your programs and updating the README, push your changes using the commands below:

```bash
git add .
git commit -m "Added new program"
git push origin main
```
