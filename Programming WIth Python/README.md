## 🤝 Como Contribuir

Se você deseja contribuir com este repositório, siga as etapas abaixo:

### ✅ Clone o repositório
```bash
git clone https://github.com/seu-usuario/repositorio-python.git
```

## ✍️ Adicione seus programas
Crie uma nova pasta ou adicione seus programas às pastas existentes (problemas_simples ou com_funcoes).

Atualize este arquivo README.md com a descrição do novo programa adicionado.

## 📤 Envie suas alterações

Após adicionar seus programas e atualizar o README, envie suas alterações com os comandos abaixo:

```bash
git add .
git commit -m "Adicionado novo programa"
git push origin main
```
