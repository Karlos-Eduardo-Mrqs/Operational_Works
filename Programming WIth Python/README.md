# 📁 Study Code Repository

This repository was created with the goal of organizing and sharing study codes in a clear and purpose-driven way. The files are divided into two main folders:

---

## 🧠 Main Folders

### 📂 `resolution_problems`

This folder contains the **solution to various problems** using algorithms and data structures. It’s ideal for those practicing programming logic or preparing for technical interviews and competitions.

#### Example contents:
- Sorting problems
- Calculations: average, sum, factorial
- Loops and conditionals
- Logical and mathematical challenges

> 📎 *Each file in this folder represents a solved problem with explanations in comments.*

---

### 📂 `work_with_classes`

This folder is dedicated to **Object-Oriented Programming (OOP)**. It showcases implementations of classes, objects, inheritance, encapsulation, polymorphism, and other related concepts.

#### Example contents:
- Creating classes in different languages
- Object simulations
- OOP exercises
- Projects using multiple classes

> 🧩 *The files usually contain practical examples of how the created classes are used.*

---

## 🛠️ How to Use This Repository

1. **Clone the repository** to your machine:
```bash
git clone https://github.com/your-username/repository-name.git
```

2. **Access the project folder**:

```bash
cd nome-do-repositorio
```

3. **Choose one of the folders to explore**:

```bash
cd resolution_problems
```

```bash
cd work_with_classes
```

4. **Run the files in the corresponding language environment.**

```bash
python nome_do_arquivo.py
````
